Documentación del Lenguaje pixelWallE

Bienvenido a la documentación del lenguaje de programación pixelWallE. Este lenguaje está diseñado específicamente para crear arte pixel a pixel de manera programática dentro del editor pixelWallE.

El lenguaje es simple y directo, permitiéndote definir formas, colores y movimientos utilizando una sintaxis clara y concisa. Su enfoque principal es la manipulación visual del lienzo mediante comandos específicos.

Esta página te ofrece una visión general del lenguaje. Para profundizar en los detalles de cada componente, navega usando la barra lateral:

  • Expresiones y Listas: Aprende sobre los tipos de datos (literales), cómo combinarlos con operadores para formar expresiones y cómo manejar colecciones de datos usando listas.
  • Comandos y Funciones: Descubre las instrucciones que le dicen a pixelWallE qué dibujar o hacer en el lienzo (comandos) y cómo obtener información sobre el estado actual del editor o del lienzo (funciones).
