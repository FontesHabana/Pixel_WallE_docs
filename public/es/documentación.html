Documentación del Editor πxelWallE: Tu Punto de Partida Creativo

¡Bienvenido a la guía esencial de πxelWallE! Aquí encontrarás todo lo que necesitas para dominar nuestro innovador editor y dar vida a tus visiones de pixel art. Hemos diseñado esta documentación para que sea tan intuitiva como nuestro editor, permitiéndote sumergirte de lleno en la creación sin complicaciones.

Desde cargar tus proyectos hasta exportar tus obras maestras y convertir imágenes en código, prepárate para una experiencia de creación de pixel art como ninguna otra.

Gestión de Archivos: Tu Creatividad, Siempre Segura

Tu progreso es invaluable. πxelWallE te ofrece herramientas claras y sencillas para cargar, guardar y organizar todos tus proyectos de pixel art.

Abrir Archivos Existentes: Retoma donde lo dejaste

Para continuar tu aventura creativa:

  • Dirígete al menú Archivo en la barra superior.
  • Selecciona Abrir....
  • Busca tu archivo .pxl (el formato nativo de πxelWallE) y haz clic para abrirlo.

¡Tu proyecto estará listo para seguir siendo transformado!

Crear Nuevos Archivos: Un Lienzo en Blanco

Para empezar de cero y dar rienda suelta a nuevas ideas:

  • En el menú Archivo, elige Nuevo.

Un editor fresco y vacío te espera, listo para tus primeras líneas de código pixel.

Guardar Archivos: ¡No Pierdas Ni un Pixel!

Guarda tu obra con frecuencia para asegurar cada avance:

  • Guardar (Ctrl + S / Cmd + S): Actualiza tu archivo existente con los últimos cambios.
  • Guardar como...: Guarda tu proyecto con un nuevo nombre o en una ubicación diferente, ideal para versiones o copias de seguridad.

Formatos de Archivo Soportados: .pxl, tu Código, tu Arte

πxelWallE utiliza el formato .pxl para almacenar el código de tu pixel art. ¡Asegúrate de usarlo para guardar y cargar tus creaciones!

Consejo Pro: Activa las copias de seguridad automáticas si la opción está disponible para una tranquilidad total.

Importar Fotos a Código: Transforma Imágenes en Creación Programable

¡Una de las características más mágicas de πxelWallE! Convierte cualquier imagen (PNG, JPG, BMP) en código pixelWallE. Es la forma perfecta de explorar la representación del pixel art programable o de empezar un nuevo proyecto a partir de una base visual existente.

Cómo Funciona: La Magia Detrás de la Conversión

Esta herramienta analiza los píxeles de tu imagen, interpretando sus colores y posiciones para generar líneas de código que, al ejecutarse, recrearán esa imagen en el editor.

Pasos para la Conversión: De la Imagen al Código

  1. Accede al menú Herramientas.
  2. Selecciona la opción Foto a Código....
  3. Elige la imagen que quieres transformar.
  4. Ajusta las opciones:
    • **Tamaño de Salida:** Define las dimensiones de tu pixel art final.
    • **Paleta de Colores:** Limita los colores para un estilo retro o específico.
    • **Umbral/Contraste:** Afina cómo los colores se interpretan.
  5. Haz clic en Convertir o Generar Código.

¡Observa cómo tu imagen se materializa en líneas de código listas para editar!

El Código Generado: Tu Nuevo Lienzo

set_pixel(x=0, y=0, color="#FF0000");
set_pixel(x=1, y=0, color="#FFA500");
# ... y así, pixel a pixel, tu imagen toma forma en código ...
draw_rectangle(x=10, y=10, width=5, height=5, color="#0000FF"); // O agrupa píxeles si la optimización lo permite

Este código es tu punto de partida. Modifícalo, experimenta y hazlo tuyo.

Consideraciones para una Conversión Óptima

  • Las imágenes más simples o con menos colores suelen generar mejores resultados.
  • Imágenes muy grandes pueden generar mucho código; experimenta con el tamaño de salida.
  • El resultado es una base sólida, pero siempre puedes refinarlo manualmente.

Consejo Pro: Juega con los ajustes de tamaño y paleta para descubrir efectos visuales sorprendentes. ¡Imágenes con contornos definidos son tus mejores aliadas!

Exportar como Imagen: ¡Comparte tu Obra Maestra Pixel!

Tu pixel art está listo para el mundo. Exporta tus creaciones a formatos de imagen estándar como PNG, JPG o BMP para compartirlas en redes, usarlas en tus proyectos o simplemente guardarlas como obras de arte digital.

Cómo Exportar tu Arte: De Código a Imagen Final

  1. Asegúrate de que la previsualización de tu pixel art es exactamente como la quieres.
  2. Dirígete al menú Archivo (o Exportar).
  3. Selecciona Exportar como Imagen....
  4. Configura las opciones de exportación:
    • **Formato de Imagen:** PNG es ideal para pixel art por su calidad sin pérdida y soporte de transparencia.
    • **Escala/Zoom:** Aumenta el tamaño de tus píxeles (ej. 2x, 4x, 8x) para que tu arte luzca perfecto en cualquier pantalla.
    • **Guardar en:** Elige la ubicación y el nombre de tu archivo.
  5. Haz clic en Exportar.

¡Listo! Tu pixel art está ahora en formato de imagen, listo para brillar.

Consejo Pro: Para la máxima nitidez y detalle en pixel art, siempre exporta a PNG y considera escalar tus píxeles al menos a 4x para una mejor visualización en pantallas modernas.

Consola Integrada: Tu Asistente de Codificación en Tiempo Real

La consola de πxelWallE es tu centro de mando. Úsala para ejecutar código, depurar errores al instante y ver la salida de tus programas, todo en un solo lugar.

Acceso a la Consola: Siempre a la Vista

Normalmente, la consola está en la parte inferior del editor. Si no la ves, actívala desde el menú Ver > Consola o Terminal.

Ejecuta y Observa: Código en Acción

Presiona el botón "Ejecutar" o escribe comandos directamente en la consola. Verás la salida de tu código y cualquier mensaje importante al instante, facilitando la depuración y el seguimiento.

Comandos Útiles (¡Explora!):

  • run tu_archivo.pxl: Ejecuta tu script de pixel art.
  • clear: Limpia la consola para una vista fresca.
  • ls (o dir): Lista tus archivos guardados.
  • run_with_args tu_script.pxl arg1 arg2: Pasa argumentos a tu código para un control dinámico.

Recuerda: La sintaxis completa está en la Documentación de la Consola.

Consejo Pro: Mantén la consola abierta. Es tu mejor amiga para entender y solucionar problemas en tu código.

Navegación Rápida: Botones Laterales para un Flujo Impecable

Descubre los atajos visuales. La barra lateral de πxelWallE te da acceso instantáneo a las funciones más usadas, haciendo tu proceso creativo más ágil y sin interrupciones.

Conoce tus nuevos accesos directos:

  • Ajustar Paneles / Pantalla Completa: / Controla el tamaño de tus paneles (código, previsualización, consola) o alterna entre modos de vista.

  • Deshacer (Undo): Vuelve un paso atrás, ¡perfecto para experimentar sin miedo!

  • Rehacer (Redo): Recupera una acción deshecha.

  • Redimensionar Elemento: (Si aplica) Permite modificar el tamaño de elementos gráficos directamente en la previsualización.

    (Nota: Esta función puede variar según la versión del editor).

  • Ejecutar Código (Run): ¡El botón mágico! Ejecuta tu código y mira tu pixel art cobrar vida.

Consejo Pro: Familiarízate con estos botones. Son tus aliados para mantener un ritmo de trabajo fluido.

Depuración en Tiempo Real: Errores, Bajo Control

πxelWallE te asiste activamente en la corrección de errores. Detecta problemas mientras escribes, dándote retroalimentación inmediata para que tu código sea perfecto.

Cómo te Ayuda πxelWallE: Señales Claras

  • Subrayado Intuitivo: Partes del código con errores se resaltan (rojo para errores, amarillo para advertencias), guiándote al problema exacto.
  • Mensajes Detallados (Tooltips): Pasa el ratón sobre el subrayado para una explicación clara del problema.
  • Panel de Problemas: Un panel dedicado lista todos los errores y advertencias, permitiéndote navegar rápidamente a cada incidencia.

Tipos de Errores que Detectamos:

  • Errores de Sintaxis: Problemas con la estructura del lenguaje (ej. un paréntesis olvidado).
  • Errores Semánticos Básicos: Cuando el código no tiene sentido lógico (ej. un color no válido).

Interpretando los Mensajes: Tu Guía para Solucionar Problemas

Cada mensaje te dirá: el tipo de problema (Error o Advertencia), una descripción clara y la ubicación exacta (línea y columna).

Consejos para una Depuración Efectiva:

  • **Lee con Atención:** El mensaje de error es tu mejor pista.
  • **Empieza por el Principio:** Un error temprano puede causar muchos problemas posteriores.
  • **Revisa Alrededor:** A veces, el problema real está en una línea cercana.
  • **Consulta la Documentación:** Siempre ten a mano la Documentación del Lenguaje pixelWallE para verificar la sintaxis.

Consejo Pro: ¡Los errores son oportunidades de aprendizaje! πxelWallE los convierte en pasos para mejorar tu código.
