Documentación del Lenguaje pixelWallE

Bienvenido a la documentación del lenguaje de programación pixelWallE. Este lenguaje está diseñado específicamente para crear arte pixel a pixel de manera programática dentro del editor pixelWallE.

El lenguaje es simple y directo, permitiéndote definir formas, colores y movimientos utilizando una sintaxis clara y concisa. Su enfoque principal es la manipulación visual del lienzo mediante comandos específicos.

Esta página te ofrece una visión general del lenguaje. Para profundizar en los detalles de cada componente, navega usando la barra lateral:

  • Expresiones y Listas: Aprende sobre los tipos de datos (literales), cómo combinarlos con operadores para formar expresiones y cómo manejar colecciones de datos usando listas.
  • Comandos (Sentencias) y Funciones: Descubre las instrucciones que le dicen a pixelWallE qué dibujar o hacer en el lienzo (comandos) y cómo obtener información sobre el estado actual del editor o del lienzo (funciones).

Vista General de la Gramática

La gramática define la estructura del lenguaje pixelWallE. Comprenderla te ayudará a escribir código sintácticamente correcto.

El código se compone de una secuencia de sentencias. Las sentencias pueden ser comandos de dibujo, declaraciones de variables, estructuras de control de flujo (si existen) o incluso expresiones que tengan efectos secundarios.

Aquí tienes una representación simplificada de la estructura del lenguaje. Las partes detalladas de Literales, Expresiones, Listas, Comandos y Funciones se explican en las páginas dedicadas.

program ::= statement*

statement ::= command_statement
          | declaration // e.g., List numbers = [...]
          | expression ; // Some expressions might be valid statements if they have side effects, like function calls
